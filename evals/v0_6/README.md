# PaperAgent v0.6 Evaluation Corpus

`cases.jsonl` currently contains a seed case for each required family. It is intentionally not yet the
48-case release corpus required by `docs/v0.6/EXECUTION_PLAN.md`.

Each case defines deterministic requirements and prohibitions. Human scientific-quality rubrics and the
external holdout digest remain pending. Missing observations are retained as failed/skipped entries in
the generated report rather than silently removed.
